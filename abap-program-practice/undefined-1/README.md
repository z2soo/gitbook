# 구매 내역 조회 프로그램 생성

## 구현 목적 및 결과

> 구매 문서 번호, 회사 코드, 구매 문서 종류 값을 입력받아 이를 가지고 EKKO 테이블에서 내역을 조회 및 데이터 변경을 할 수 있는 프로그램을 작성해본다. 자세한 설명이 생략된 부분은 이전 인사 프로그램 생성 부분을 참조하도록 한다.
