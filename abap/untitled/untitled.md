# Table

## 1. Table 종류 및 속성

T-code SE11에서 테이블을 생성하여 활성화하면, 데이터베이스에서 물리적인 테이블이 생성된다. ABAP 테이블은 3 종류 존재하며, 테이블 기본 속성은 DD02L 테이블, 테이블 필드에 대한 정보는 DD03L 테이블에 저장된다.

* Transparent table
* Pooled table
* Cluster table
